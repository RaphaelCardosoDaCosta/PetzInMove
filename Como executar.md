# Como Executar

## Instalação de Dependências

### Backend
```powershell
cd backend
npm install
```

### Frontend
```powershell
npm install
```

---

## Executar o Projeto

### Terminal 1 - Backend
```powershell
cd backend
npm run dev
```

### Terminal 2 - Frontend
```powershell
npm run dev
```

---

## Opcional - Prisma Studio (Visualizar Dados)
```powershell
cd backend
npx prisma studio
```

Acesse: http://localhost:51212 (ou a porta mostrada no terminal)

---

## URLs

- **Frontend**: http://localhost:5173 (ou porta mostrada no terminal)
- **Backend API**: http://localhost:5000
- **Prisma Studio**: http://localhost:51212 (quando rodando)
